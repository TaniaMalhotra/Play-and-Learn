HTML is basic fundamental building block of web pages.It contains the tags required to display basic elements on web pages.. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript). . CSS is basic styling block of web pages.It contains the tags required to organise basic elements on web pages. It defines the design and layout of web content. Other technologies besides CSS are generally used to describe a web page's layout or functionality/behavior (JavaScript). JS is basic fundamental functionality block of web pages.It contains the functions required to organise basic elements on web pages. It defines the method and functions of web content. Other technologies besides JS are generally used to describe a web page's layout or basic structure or framework.
